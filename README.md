# devharsh.me

Source for my personal website, **[www.devharsh.me](https://www.devharsh.me)**.

A fast static site built with plain HTML and CSS (no build step), hosted on GitHub Pages. It covers my research and publications, industry and research experience, teaching, and open-source projects.

## Run locally

```bash
python3 -m http.server 8000   # then open http://localhost:8000
```

## Structure

- Pages: `index.html`, `research.html`, `experience.html`, `teaching.html`, `projects.html`, `blog.html`, `cv.html`, `404.html`
- `assets/` — stylesheet, profile image, and CV
- `CNAME`, `robots.txt`, `sitemap.xml`
